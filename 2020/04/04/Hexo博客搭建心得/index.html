Hexo博客搭建心得
Created 2020-04-04| Updated 2020-04-07|学习笔记
Post View:

前期准备工作

1.有一个github账号，没有的话去注册一个；
2.安装了node.js、npm，并了解相关基础知识；
3.安装了git

安装Hexo

1.全局安装Hexo

$ npm install -g hexo

2.初始化Hexo博客

在电脑的某个地方新建一个名为blog的文件夹（名字可以随便取），比如我的是D:\blog，由于这个文件夹将来就作为你存放代码的地方，所以最好不要随便放。

$ cd /d/blog/   # 进入blog文件夹
$ hexo init   # 初始化博客
$ hexo g   # 生成静态页面
$ hexo s   # 启动本地服务，一般是4000端口

修改Hexo主题

Hexo的官方主题比较丑，这样的话肯定影响我们写代码的心情，那怎么办呢？？
不必惊慌，让我们来给它改皮换面，换上一个我们心水的主题。。
博主在这里推荐   hexo-theme-yilia。

先把这份主题下载下来：

$ git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

下载后，在blog目录下找到 _config.yml ,并将其中的 theme:landscape 修改为 theme:yilia。
然后执行：
hexo c  (清理public的内容);
hexo g & hexo s  (重新生成和发布)

发布到Github

创建仓库

新建一个名为你的用户名.github.io的仓库，比如说，如果你的github用户名是test，那么你就新建test.github.io的仓库（必须是你的用户名，其它名称无效），将来你的网站访问地址就是 http://test.github.io 了。

上传Github
