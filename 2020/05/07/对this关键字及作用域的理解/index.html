this 的绑定形式

在 JavaScript 中，this 的指向有 4 种绑定形式

  1. 默认绑定（指向 window）

    直接调用函数，this 指向 window

  2. 隐式绑定

    隐式绑定，this 指向调用者对象

  3. 硬绑定

    通过 call, apply 来改变 this 的指向

  4. 构造函数绑定
